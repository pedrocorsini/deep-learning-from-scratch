# Deep Learning from Scratch

A hands-on learning repository implementing deep learning concepts from first principles using Python and NumPy.

## Overview

This project explores fundamental deep learning concepts through practical examples and implementations. It serves as a learning resource for understanding how neural networks work at their core, without relying on high-level frameworks.

## Structure

```
deep-learning-from-scratch/
├── examples/
│   ├── arrays.py                          (NumPy operations and activation functions)
│   └── derivatives.py                     (Numerical differentiation)
├── LICENSE
└── README.md
```

## Getting Started

### Prerequisites

- Python 3.x
- NumPy

### Running Examples

```bash
# Array operations and activation functions
python examples/arrays.py

# Numerical derivatives
python examples/derivatives.py
```

## Topics Covered

- Array operations with NumPy
- Activation functions (ReLU, Leaky ReLU)
- Derivatives and numerical differentiation
- Foundational neural network concepts
